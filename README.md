## ✨ 프로젝트 소개

- AI, 드로잉, 이미지를 활용해 나만의 에셋을 제작할 수 있어요.
- 제작한 에셋을 바탕으로 프레임을 만들어보세요.
- 제작한 프레임은 NFT로 저장되어 나만의 고유한 프레임으로 간직할 수 있어요.
- 촬영한 사진을 프레임에 넣어 갤러리에 추억을 저장하세요.
